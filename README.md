# OrangeHRM Automation Testing Project

## 📌 Project Description
This project automates key functionalities of the OrangeHRM web application using Selenium WebDriver with Java.  
The goal of this project is to practice automation testing concepts such as Page Object Model (POM), TestNG framework, and reusable utilities.

---

## 🛠 Tools & Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- IntelliJ IDEA
- Git & GitHub

---

## 📋 Test Scenarios Automated
- Login functionality (valid login)
- Login validation (invalid credentials)
- Leave request submission (simulated claim submission)
- End-to-end workflow testing
- Screenshot capture on test failure

---

## 🧩 Framework Features
- Page Object Model (POM) design pattern
- Wait utility for handling dynamic elements
- Screenshot utility for capturing failures
- TestNG test execution
- Maven dependency management
